# python_progs

1. plot_nbody.py
Plotting trajectories of N bodies under gravitational interaction.
Analytical solutions generated by nbody.cc
Usage of program:
python3 plot_nbody.py X
where X = 0, 1, 2, 3, 4, 5 ,6
X = 0, Three bodies of arbitrary mass on vertices of equilateral triangle
X = 1,2,3 Three bodies of arbitrary mass aligned
X = 4, 5  Four bodies on vertices of isosceles trapezoid
X = 6  Six bodies with equal mass on vertices of regular polygon
./output contains .png figures with trajectories
